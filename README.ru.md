# Электроника МК85
### Прочитайте это на других языках: [English](README.md), [Русский](README.ru.md).
## Программы

  [1) SQR_EQ.BAS](src/SQR_EQ.BAS)
  >Программа для нахождения корней квадратного уравнения. Прямой ввод уравнения (встроенный парсер).
  >Уравнения необходимо вводить без пробелов.
  >
  >🟢Отлично работает на базовой версии микрокомпьютера.

  [2) D2Any.BAS](src/D2Any.BAS)
  >Программа переводит десятичное число "d" в число с основанием "m" (от 2 до 36)
  >
  >🟢Works perfectly on the basic version of the microcomputer.

## Игры

  [1) BULLSnCOWS.BAS](src/BULLSnCOWS.BAS)
  >Классическая игры "Быки и Коровы"
  >
  >Компьютер задумывает 4 значное число и вам нужно угадать задуманные компьютером числа.
  >
  >🟢Отлично работает на базовой версии микрокомпьютера.
